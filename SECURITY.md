# Security Policy

## Supported Versions

As this is a research framework, we currently support the latest published iteration of the GHDP documentation.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Foresight 88 Institute takes the integrity of our macro-structural data seriously.

If you discover a vulnerability or a data integrity issue within our framework, please do not disclose it publicly. Instead, please report it confidentially to our research lead.

**Email:** eunice.wong@foresight88.institute

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress.
